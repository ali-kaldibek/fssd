@font-face ережесі авторларға веб-беттердегі мәтінді көрсету үшін онлайн қаріптерді көрсетуге мүмкіндік береді.

Авторларға өз қаріптерін ұсынуға мүмкіндік бере отырып, @font-face пайдаланушылар өздерінің компьютерлерінде орнатқан қаріптердің шектеулі санына байланысты болу қажеттілігін жояды.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@font-face {
    font-family: 'Open Sans';
    src: url('/fonts/OpenSans-Regular-webfont.woff2') format('woff2'),
      url('/fonts/OpenSans-Regular-webfont.woff') format('woff');
}

Rule Descriptors

font-display
Қаріптің жүктелуіне және пайдалануға дайын болуына байланысты оны көрсету әдісін анықтайды.
font-family
Қаріп қасиеттері үшін қаріп мәні ретінде пайдаланылатын атауды көрсетеді.
font-stretch
font-stretch мәні.
font-style
font-style мәні.
font-weight
font-weight мәні.
font-variant
font-variant мәні.
font-feature-settings
OpenType қаріптерінде кеңейтілген типографиялық мүмкіндіктерді басқаруға мүмкіндік береді.
font-variation-settings
OpenType немесе TrueType қаріп деңгейін төмен деңгейде басқаруға мүмкіндік береді, олардың мәндерімен бірге өзгеруі мүмкін төрт әріптік функция атауларының аттарын көрсетеді.
src
Қаріп деректері бар ресурсты орнатады. Бұл қаріп файлының URL мекенжайы немесе пайдаланушының компьютеріндегі қаріп атауы болуы мүмкін.
unicode-range
Қаріптен қолданылатын Юникод кодтарының ауқымы.
