Псевдо элемент ::after таңдалған элементтің виртуалды соңғы еншілесіне сәйкес келеді.

Ол әдетте мазмұнның CSS сипатын пайдаланып элементке косметикалық мазмұн қосу үшін қолданылады. Әдепкі бойынша бұл элемент inline

@rules

::after ::backdrop ::before ::cue ::first-letter ::first-line ::grammar-error ::marker ::part() ::placeholder ::selection ::slotted() ::spelling-error

Syntax

selector::after{
    declaration
}
