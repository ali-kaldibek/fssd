@keyframes ережесі CSS анимация тізбегіндегі аралық қадамдарды басқарады, анимация тізбегінің негізгі кадрларының стильдерін анықтайды.

Бұл ауысуларға қарағанда анимация тізбегінің аралық кезеңдерін көбірек басқаруға мүмкіндік береді.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@keyframes slidein {
    from {
      margin-left: 100%;
      width: 300%;
    }
  
    50% {
      margin-left: 50%;
      width: 200%;
    }
  
    to {
      margin-left: 0%;
      width: 100%;
    }
}
