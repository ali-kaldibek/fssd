@import ережесі CSS мәнерлерін басқа стиль кестелерінен импорттау үшін қолданылады.

Бұл ережелер @charset ережелерінен басқа Ережелердің барлық түрлерінен бұрын болуы керек, өйткені бұл кірістірілген оператор емес, @import шартты топтық at ережелерінде қолданыла алмайды.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@import url('fineprint.css') print;
@import url('bluish.css') projection, tv;
@import 'custom.css';
@import url('chrome://communicator/skin/');
@import 'common.css' screen, projection;
@import url('landscape.css') screen and
(orientation: landscape);

Мәні

Браузерлер қолдау көрсетілмейтін медиа түрлеріне ресурстарды жүктемеуі үшін авторлар @import медиа-тәуелді ережелерін көрсете алады. Бұл шартты импорттаушылар URL мекен-жайынан кейін үтірмен бөлінген медиа сұрауларын орнатады.

Қандай да бір медиа-сұрау болмаған жағдайда импорт сөзсіз болып табылады. All көрсеткіші бірдей әсер етеді.
