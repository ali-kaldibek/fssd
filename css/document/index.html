@document ережесі құжаттың URL мекен-жайына негізделген стиль ережелерін шектейді. Ол негізінен жеке стиль кестелеріне арналған, бірақ оны авторлық стиль кестелерінде де қолдануға болады.

Мысалы, егер сайттың мобильді және негізгі нұсқаларының мекен-жайлары әртүрлі болса, онда біз әр нұсқа үшін элементтердің өзіндік стилін орната аламыз.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@document url("https://www.example.com/")
{
  h1 {
    color: green;
  }
}
