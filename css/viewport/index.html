@viewport ережесі әр түрлі құрылғыларға және олардың өлшемдеріне байланысты веб-беттің орналасуын оңтайландыруға мүмкіндік береді. Бұл <meta name="viewport"> аналогы, бірақ параметрлерді HTML-ден CSS ережелеріне ауыстырады.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@viewport {
  min-width: 640px;
  max-width: 800px;
}
@viewport {
  zoom: 0.75;
  min-zoom: 0.5;
  max-zoom: 0.9;
}
@viewport {
  orientation: landscape;
}
