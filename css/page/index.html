@page ережесі құжатты басып шығару кезінде CSS-тің кейбір қасиеттерін өзгерту үшін қолданылады. @page көмегімен барлық CSS сипаттарын өзгерте алмайсыз. Сіз тек margin, orphans, widows және құжат бетінің үзілістерін өзгерте аласыз. CSS-тің басқа қасиеттерін өзгерту әрекеттері еленбейді.

@page ережесі CSS CSSPageRule нысан моделінің интерфейсі арқылы қол жетімді.

@rules and modifiers

@charset @import @namespace @media @supports @document @page @font-face @keyframes @viewport @counter-style @font-feature-values

Syntax

@page {
    margin: 1cm;{
  }
  
  @page :first {
    margin: 2cm;
  }
}
size
Блоктағы бет өрістерінің мақсатты өлшемі мен бағытын көрсетеді. Жалпы жағдайда, парақтың бір өрісі парақтың бір парағында көрсетілген кезде, ол парақтың өлшемі мен бағытын анықтайды.

Syntax

/* Keyword values for scalable size */
size: auto;
size: portrait;
size: landscape;

/* <length> values */
/* 1 value: height = width */
size: 6in;

/* 2 values: width then height */
size: 4in 6in;

/* Keyword values for absolute size */
size: A4;
size: B5;
size: JIS-B4;
size: letter;

/* Mixing size and orientation */
size: A4 portrait;

Мәні

Әдетті мәні
auto
Браузер парақтың өлшемін өзі таңдайды.
landscape
Беттің мазмұны ландшафт бағдарында көрсетіледі
portrait
Беттің мазмұны портрет режимінде көрсетіледі. Бұл әдепкі бағдар.
<length>
Кез келген ұзындық мәні. Бірінші мән беттің жиегінің еніне сәйкес келеді, ал екіншісі оның биіктігіне сәйкес келеді. Егер тек бір мән көрсетілсе, ол ені мен биіктігі үшін де қолданылады.
<page-size>
A5 (148mm x 210mm), A4 (210mm x 297mm), A3 (297mm x 420mm), B5 (176mm x 250mm), B4 (250mm x 353mm), JIS-B5 (182mm x 257mm), JIS-B4 (257mm x 364mm), letter (8.5in x 11in), legal (8.5in x 14in), ledger (11in x 17in).
Әдепкі бойынша бұл мән өшірулі.
marks
Құжатқа кесу және / немесе тіркеу белгілерін қосады.

Syntax

@page {
    /* Keyword values */
    marks: none;
    marks: crop;
    marks: cross;
    marks: crop cross;
}

Мәні

Әдетті мәні
crop
Кесуге арналған белгілер басылады
cross
Кресттер басып шығарылады
none
Ешқандай белгілер басылмайды
Әдепкі бойынша бұл мән өшірулі.
bleed
Беттің көрсетілетін аймағы кесілетін беттің шетінен тыс түсу дәрежесін анықтайды.

Syntax

/* Keyword values */
bleed: auto;

/* <length> values */
bleed: 8pt;
bleed: 1cm;

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.
