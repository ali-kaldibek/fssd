</button> Tag ( button — "батырма" )
веб-бетте түймелер жасайды және оның әрекеті <input> көмегімен алынған нәтижеге ұқсайды (type="button | reset | submit"атрибутымен).

Бұл элементтен айырмашылығы, </button> батырмаларды құрудың кеңейтілген мүмкіндіктерін ұсынады. Мысалы, ұқсас батырмада кез-келген HTML элементтерін, соның ішінде суреттерді орналастыруға болады. Стильдерді қолдана отырып, шрифт, өң түсі, өлшемдері және басқа параметрлерді өзгерту арқылы батырманың көрінісін анықтауға болады.

</tag> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


autofocus
autofocus атрибуты бет жүктелгеннен кейін түйме фокусты алатындығын орнатады. Мұндай түймені оған назар аудармай бірден басуға болады, мысалы, пернетақтаны пайдалану.

Syntax

<button autofocus>
    ...
</button>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


disabled
Түймеге кіруді және оны өзгертуді блоктайды. Бұл жағдайда ол сұр түспен көрсетіледі және пайдаланушы іске қосу үшін қол жетімді емес. Сонымен қатар, мұндай батырма Tab пернесін, тінтуірді немесе басқа әдісті басу арқылы фокусты ала алмайды. Дегенмен, батырманың бұл күйін сценарийлер арқылы өзгертуге болады. Құлыпталған батырманың мәні серверге берілмейді.

Syntax

<button disabled>
    ...
</button>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


form
Түймені оның идентификаторы бойынша формамен байланыстырады. Мұндай байланыс түйме form элементінің ішінде орналаспаған жағдайда қажет, мысалы, оны бағдарламалық түрде жасаған кезде.

Syntax

<button form="мәні">
    ...
</button>

Мәні

Әдетті мәні
Пішін идентификаторы (form элементінің id төлсипатының мәні). Жоқ.


formaction
Пішін өңдегішінің мекен — жайын анықтайды-бұл пішін деректерін алатын және олармен қажетті әрекеттерді жасайтын бағдарлама. Formaction атрибутының әрекеті form элементінің action атрибутына ұқсас. Егер сіз action және form action бір уақытта көрсетсеңіз, батырманы басқан кезде action атрибуты еленбейді және деректер formaction-да көрсетілген мекен-жайға жіберіледі.

Syntax

<button formaction="<адрес>">
    ...
</button>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


formenctype
Серверге жіберілген кезде форма деректерін кодтау әдісін орнатады. Әдетте файлды жіберу үшін өріс қолданылған жағдайда нақты көрсетіледі (input type="file"). Бұл атрибут form элементінің enctype атрибутына ұқсас.

Syntax

<button formenctype="мәні">
    ...
</button>

Мәні

Әдетті мәні
application/x-www-form-urlencoded
Бос орындардың орнына+, орыс әріптері сияқты таңбалар он алтылық мәндермен кодталады (мысалы, %D0%9F%D0%B5%D1%82%D1%8F Петяның орнына).
multipart/form-data
Деректер кодталмайды. Бұл мән файлдарды жіберу кезінде қолданылады.
text/plain
Бос орындар + белгісімен ауыстырылады, әріптер мен басқа таңбалар кодталмайды.
application/x-www-form-urlencoded


formmethod
Атрибут браузерге форма деректерін серверге қандай әдіспен жіберу керектігін айтады.

Syntax

<button formmethod="мәні">
    ...
</button>

Мәні

Әдетті мәні
GET
Бұл әдіс сұрау белгісінен кейін беттің мекен-жайына қосылатын және амперсандпен (&символы) бөлінетін «"атау=мән"» жұбы түрінде мекен-жай жолына тікелей форма деректерін жіберуге арналған. Мысалы, толық мекен-жайы болады http://site.ru/doc/?name=Vasya&password=pup. әдістегі деректер көлемі 4 Кб-мен шектелген.
POST
Серверге браузер сұрауында деректерді жібереді, жіберілетін деректердің көлемі тек сервер баптауларымен шектеледі.
Жоқ.

formnovalidate
Түймені басқан кезде пайдаланушы енгізген деректерді дұрыстылық түрінде тексеруді болдырмайды. Мұндай тексеруді <input type="email">, <input type="url"> өрістері үшін пішінді серверге жіберген кезде, сондай-ақ <input> элементінде pattern немесе required төлсипаты болған кезде браузер автоматты түрде жасайды.

Syntax

<button formnovalidate>
    ...
</button>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.



formtarget
HTML құжаты ретінде пішін өңдегіші қайтарған нәтиже жүктелетін кадрдың атын анықтайды.

Syntax

<button formtarget="мәні">
    ...
</button>

Мәні

Әдетті мәні
_blank
Бетті жаңа шолғыш қойындысына жүктейді.
_self
Бетті ағымдағы қойындыға жүктейді.
_parent
Бетті ата-ана жақтауына жүктейді; егер жақтаулар болмаса, онда бұл мән _self ретінде жұмыс істейді.
_top
Барлық Жақтауларды болдырмайды және бетті толық шолғыш терезесінде жүктейді; егер жақтаулар болмаса, онда бұл мән _self ретінде жұмыс істейді.
Жоқ.


name
Батырманың бірегей атауын анықтайды. Әдетте, бұл атау батырманың мәнін серверге жібергенде немесе сценарийлер арқылы түймеге кіру үшін қолданылады.

Syntax

<button name="мәні">
    ...
</button>

Мәні

Әдетті мәні
Атау ретінде таңбалар жиынтығы, оның ішінде сандар мен әріптер қолданылады. JavaScript регистрге сезімтал, сондықтан аты бойынша элементпен байланысқан кезде name атрибутындағыдай жазу формасын сақтаңыз. Жоқ.


type
Оның мінез-құлқын формада белгілейтін түйме түрін анықтайды. Сыртқы көріністе әр түрлі типтегі түймелер әр түрлі болмайды, бірақ мұндай батырмалардың әрқайсысының өз функциялары бар.

Syntax

<button type="мәні">
    ...
</button>

Мәні

Әдетті мәні
button
Кәдімгі түйме.
reset
Енгізілген пішін деректерін тазартуға және мәндерді бастапқы күйіне қайтаруға арналған батырма.
submit
Пішін деректерін серверге жіберуге арналған батырма.
menu
menu элементі арқылы жасалған мәзірді ашады.
submit


value
Серверге жіберілетін батырманың мәнін анықтайды. «атау=мән» жұбы жіберіледі, онда атау button элементінің name төлсипатымен, ал мәны value төлсипатымен беріледі. Мән батырмадағы мәтінмен сәйкес келуі мүмкін немесе тәуелсіз болуы мүмкін. Сондай-ақ, value атрибуты сценарийлер арқылы деректерге қол жеткізу үшін қолданылады.

Syntax

<button value="мәні">
    ...
</button>

Мәні

Әдетті мәні
Кез келген мәтін жолы. Жоқ.
