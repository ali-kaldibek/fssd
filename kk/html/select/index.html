</select> Tag ( select — "таңдау" )
ашылмалы тізім ретінде интерфейс элементін, сонымен қатар бір немесе бірнеше таңдау тізімін жасауға мүмкіндік береді.

Соңғы көрініс тізімнің биіктігін белгілейтін size төлсипатын пайдалануға байланысты. Тізімнің ені </option> элементінде көрсетілген ең кең мәтінмен анықталады және стильдер арқылы да өзгеруі мүмкін. Әр элемент </select> элементінің көмегімен жасалады, оны </select> контейнеріне салу керек. Егер сіз тізім деректерін серверге жіберуді жоспарласаңыз, онда пішіннің ішіне </select> қою керек. Бұл тізім деректеріне сценарийлер арқылы жүгінген кезде де қажет.

</select> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


autofocus
autofocus төлсипаты бетті жүктегеннен кейін тізім фокусты алады, мысалы, пернетақта арқылы элементтерді таңдау үшін тізім қол жетімді болады.

Syntax

<select autofocus>
    ...
</select>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


disabled
Тізім элементтеріне кіруге және өзгертуге тыйым салады. Бұғатталған тізім курсор немесе пернетақта арқылы фокусты ала алмайды, өзгертілмейді, мұндай тізімнің мәні серверге берілмейді.

Syntax

<select disabled>
    ...
</select>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


form
Тізімді оның идентификаторы бойынша формамен байланыстырады. Мұндай байланыс тізім <form> шегінен тыс орналасқан жағдайда қажет.

Syntax

<select form="мәні">
    ...
</select>

Мәні

Әдетті мәні
Пішін идентификаторы (<form> элементінің id төлсипатының мәні). Жоқ.

multiple
multiple төлсипатының болуы шолғышқа </select> элементінің мазмұнын бірнеше таңдау тізімі ретінде көрсетуді хабарлайды. Тізімнің соңғы түрі қолданылатын size атрибуты мен шолғышқа байланысты.
Бірнеше тізім мәндерін таңдау үшін тінтуір курсорымен бірге Ctrl және Shift пернелері қолданылады.
Деректер массивін серверге жіберу үшін name төлсипатының мәні төртбұрышты жақшалармен жазылуы керек — мысалы, hero[ ].

Syntax

<select multiple>
    ...
</select>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


name
</select> элементінің бірегей атауын анықтайды. Әдетте, бұл атау сценарийлер арқылы деректерге қол жеткізу немесе серверде таңдалған тізім мәнін алу үшін қолданылады.

Syntax

<select name="мәні">
    ...
</select>

Мәні

Әдетті мәні
Атау ретінде таңбалар жиынтығы, оның ішінде сандар мен әріптер қолданылады. JS регистрге сезімтал, сондықтан </select> элементіне атау атрибутымен бірдей жазу формасын сақтаңыз. Жоқ.

required
Пішінді серверге жібермес бұрын таңдау үшін міндетті тізімді орнатады. Егер тізім элементі таңдалмаса, шолғыш хабарлама шығарады және форма жіберілмейді. Хабарламаның түрі мен мазмұны браузерге байланысты және пайдаланушы өзгерте алмайды.

Syntax

<select required>
    ...
</select>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша, required атрибут өшірулі.


size
Тізімнің биіктігін орнатады. Егер size төлсипатының мәні 1 болса, онда тізім ашылғышқа айналады. size="1" кезінде </select> элементіне multiple төлсипатын қосқан кезде тізім "бұралу" ретінде көрсетіледі. Барлық басқа жағдайларда бір немесе бірнеше таңдау тізімі алынады.

Syntax

<select size="мәні">
    ...
</select>

Мәні

Әдетті мәні
Кез келген бүтін оң Сан. multiple атрибутына байланысты. Егер ол бар болса, онда тізім мөлшері элементтер санына тең болады. multiple төлсипаты болмаған кезде, size төлсипатының әдепкі мәні 1 болады.
