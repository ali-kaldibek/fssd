</label> Tag ( label — "белгі, төте жол" )
әдетте мәтін ретінде әрекет ететін белгілі бір белгі мен пішін элементі (<input>, </select>, </textarea>) арасындағы байланысты орнатады.

Мұндай байланыс мәтінді тінтуірмен басқан кезде пішін элементтерінің мәндерін өзгерту үшін қажет. Сонымен қатар, </label> көмегімен пернетақтада жылдам пернелерді орнатуға және сілтемелер сияқты белсенді элементке өтуге болады.

Нысанды және белгіні байланыстырудың екі әдісі бар. Біріншісі-пішін элементінің ішінде id идентификаторын пайдалану және оның атын </label> элементінің атрибуты ретінде көрсету. Екінші әдіспен пішін элементі </label> контейнерінің ішіне орналастырылады.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


for
Байланыс орнатылатын <input> элементінің id төлсипаты арқылы анықталатын бірегей идентификаторды орнатады. for төлсипаты пішін элементі мен мәтін бөлінген жағдайда орнатылуы керек. Егер <input> </label> контейнерінің ішіне орналастырылса, онда for төлсипатын қолдану талап етілмейді.

Syntax

<label for="идентификатор">
    ...
</label>

Мәні

Әдетті мәні
Идентификатордың аты. Мұндай атау регистрге сезімтал, сондықтан оны <input> элементінің ішінде сипатталғандай жазу керек. Жоқ.
