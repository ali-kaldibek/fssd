<!DOCTYPE html> Declaration

<!DOCTYPE> Tag "document type" — «құжат түрі»

<!DOCTYPE> Tag ( document type — "құжат түрі" )
браузерге HTML нұсқасын көрсетеді. HTML-дің ағымдағы, бесінші нұсқасын көрсету үшін былай жазу керек: <!DOCTYPE html>.

Doctype тегі HTML тегінің алдында құжаттың басында жазылады. Тег міндетті болып табылады, бірақ бет онсыз жұмыс істейді, бірақ бұл жағдайда кейбір заттар doctype сияқты жұмыс істемейді. Сондықтан оны әрдайым жазыңыз - кейбір мәселелерден аулақ болыңыз.
