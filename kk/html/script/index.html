сценарийлерді сипаттауға арналған, бағдарламаға немесе оның мәтініне белгілі бір тілде сілтеме болуы мүмкін.

Syntax

</script> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


async
Егер async атрибуты болса, шолғыш сценарийді асинхронды түрде іске қосады. Бұл src атрибутында көрсетілген файл веб-бетті жүктеуді және көрсетуді күтпестен орындалатынын білдіреді. Сонымен қатар, бет сценарийдің нәтижесін күтпейді, бірақ әдеттегідей жүктеуді жалғастырады.
Егер бірден екі атрибут орнатылса — async және defer, онда async атрибуты еленбейді.

Syntax

<script async>
...
</script>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша, бұл атрибут өшірулі.


defer
<defer> атрибуты бүкіл бет толығымен жүктелгенге дейін сценарийдің орындалуын кейінге қалдырады. <src> атрибуты болған кезде ғана жұмыс істейді.

Syntax

<script src="мәні" defer>
    ...
</script>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша, бұл атрибут өшірулі.


src
Сыртқы файлдан сценарийі бар файлды жүктейді. Сыртқы нұсқаулар толығымен жүктелгеннен кейін, браузер оларды ағымдағы құжатқа енгізілгендей өңдейді. JavaScript сценарийлері бар файлдарда әдетте js кеңейтімі бар және оларды кез-келген жерде орналастыруға болады.
Сыртқы файлдарды қолданудың бірнеше артықшылығы бар. Бірінші қоңыраудан кейін сыртқы файл браузер кэшінде сақталады және келесі қоңыраулар тезірек жүктеледі. Сонымен қатар, шақырылған функцияларды бір жерде басқару ыңғайлы және кодты басқару оңай. Бұл, әсіресе, көптеген веб-беттер бірдей сценарийлерді қолданған кезде дұрыс.

Syntax

<script src="мәні">
    ...
</script>

Мәні

Әдетті мәні
Файлға толық немесе салыстырмалы жол мән ретінде қабылданады. Файл кеңейтімге ие болуы керек .js. Жоқ.


type
Белгілі бір тілдің MIME - түрін көрсетеді.

Syntax

<script type="мәні">
    ...
</script>

Әдетті мәні
