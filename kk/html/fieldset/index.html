</fieldset> Tag ( field set — "өрістер жиынтығы" )
пішін элементтерін топтастыруға арналған.

Мұндай топтау көптеген деректерді қамтитын формалармен жұмыс істеуді жеңілдетеді, мысалы, бір блок мәтіндік ақпаратты енгізуге, ал екіншісі құсбелгілерге арналған болуы мүмкін.

Көрнекілікті арттыру үшін браузерлер </fieldset> элементін жақтау ретінде пайдалану нәтижесін көрсетеді. Оның көрінісі операциялық жүйеге, сондай-ақ қолданылатын браузерге байланысты.

</fieldset> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


disabled
fieldset тегінің ішінде орналасқан пішін элементтеріне кіруді бұғаттайды. Пішін өрістері олардың әрқайсысына disabled атрибуты қосылғандай көрінеді.

Syntax

<fieldset disabled>
    ...
</fieldset>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


form
fieldset тобын оның идентификаторы бойынша формамен байланыстырады. Мұндай байланыс элемент form ішінде орналаспаған жағдайда, мысалы, оны бағдарламалық түрде жасаған кезде қажет. Егер байланыстыру form және fieldset өзара орнатылса, онда сіз деректерді серверге жібере аласыз және пішін ішіндегі элементтер сияқты жұмыс жасай аласыз.

Syntax

<fieldset form="мәні">
    ...
</fieldset>

Әдетті мәні
