</body> Tag ( body — "дене" )
браузер терезесінде көрсетілетін веб-беттің (мазмұнын) сақтауға арналған.

Құжатта көрсетілетін ақпаратты дәл </body> контейнерінің ішіне орналастыру керек. Мұндай ақпаратқа мәтін, суреттер, тегтер, JavaScript сценарийлері және т. б.

</body> сонымен қатар Веб-беттегі сілтемелер мен мәтіннің түстерін анықтау үшін қолданылады. HTML-де мұндай тәжірибе айыпталады және оның орнына түс схемасын көрсету үшін стильдерді оларды /body селекторына қолдану ұсынылады.
