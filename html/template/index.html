</template> Tag ( template — "шаблон" )
бұл бетті жүктеу кезінде көрінбейтін, бірақ кейінірек JS-пен толтырылуы мүмкін клиент жағындағы мазмұнды сақтау механизмі.

</template> мазмұны HTML фрагменті үшін шаблон болып табылады, оны сценарийлер арқылы құжатқа клондауға және енгізуге болады. Ол әдетте тізімдер, кестелер, </select> тізімдері және т. б. сияқты қайталанатын құрылымы бар элементтер үшін қолданылады.
