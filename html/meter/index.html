</meter> Tag ( meter — "есептегіш, өлшеуіш" )
белгілі бір диапазонда мәнді шығару үшін қолданылады.

Ол негізінен сандық мәндерді көрсету үшін қолданылады, мысалы, іздеу нәтижелерінің саны, сұйықтық көлемі, қысым және т. б.

</meter> элементін қолдайтын браузерлер мәнді көрсетілген диапазондағы ағымдағы мән түспен белгіленген үлгі ретінде көрсетеді. Қалған браузерлер </meter> ішінде жазылған мәтінді көрсетеді.

</meter> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


value
Өлшем бірлігінің мәнін белгілейді. Мән min және max атрибуттарымен белгіленген диапазонда болуы керек.

Syntax

<meter value="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. Теріс мәндерді қолдануға рұқсат етіледі. Жоқ.


min
Ол мәнге жететін минималды шекті орнатады.

Syntax

<meter value="мәні" min="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. min ≤ value ≤ max шарты міндетті түрде орындалуы керек. 0


max
Ол мәнге жететін максималды шекті орнатады.

Syntax

<meter value="мәні" max="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. min ≤ value ≤ max шарты міндетті түрде орындалуы керек. 1


low
Ол шекті анықтайды, оған жеткен кезде мән төмен болып саналады.

Syntax

<meter value="мәні" low="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. min ≤ low ≤ max, сондай-ақ low ≤ high жағдайы сақталуы керек. Жоқ.


high
Мән жоғары болып саналатын шекті анықтайды.

Syntax

<meter value="мәні" high="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. min ≤ high ≤ max, сондай-ақ low ≤ high жағдайы сақталуы керек. Жоқ.


optimum
Ең жақсы немесе оңтайлы мәнді анықтайды.

Syntax

<meter value="мәні" optimum="мәні">
    ...
</meter>

Мәні

Әдетті мәні
Мән ретінде бүтін немесе бөлшек сан көрсетіледі. min ≤ optimum ≤ max шарты міндетті түрде сақталуы керек. Жоқ.
