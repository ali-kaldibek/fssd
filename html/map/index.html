</map> Tag ( map — "карта" )
кескін карталарының белсенді аймақтарын анықтайтын <area> элементтеріне арналған контейнер ретінде қызмет етеді.

Мұндай аймақтар HTML құжаттарына сілтемелер болып табылатын суретте көрінбейтін аймақтарды орнатады. </map> пайдалану мақсаты <img> элементін клиенттік кескін картасымен байланыстыру болып табылады. Бұл байланыс usemap төлсипатымен берілген <img> - де де, name төлсипатымен орнатылған </map> - де де бір Идентификаторды қолдану арқылы анықталады.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.
name
сурет-Картаны бірегей анықтайтын идентификатор атауы. </map> элементінің name төлсипатының мәні usemap ішіндегі атқа сәйкес келуі керек. Бұл жағдайда usemap төлсипатының мәні <img> тор таңбасынан басталады.

    ...
</map>
