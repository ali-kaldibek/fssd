<link> Tag ( link — "сілтеме, байланыс" )
стильдері немесе қаріптері бар файл сияқты сыртқы құжатпен байланыс орнатады.

<link> элементі әдетте </head> контейнерінің ішіне орналастырылады және </a> элементіне қарағанда сілтеме жасамайды.

<link> жұпсыз tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


href
Сілтеме жасалатын файлға жол.

Syntax

<link href="мәні">

Мәні

Әдетті мәні
Файлға толық немесе салыстырмалы жол мән ретінде қабылданады.
мекен-жайы 		Жоқ.


media
Стильді дизайнды қолдану керек құрылғыны анықтайды. Бұл құжатты монитор экранында және басып шығару кезінде көрсету үшін әртүрлі стиль жасауға мүмкіндік береді. Үтір арқылы бірнеше мән жазуға рұқсат етіледі.

Syntax

<link media="мәні">

Мәні

Әдетті мәні
all
Барлық құрылғылар.
print
Принтер сияқты басып шығару құрылғысы.
screen
Монитор экраны.
speech
Сөйлеу синтезаторлары, сондай-ақ мәтінді дауыстап шығаруға арналған бағдарламалар. Бұған сөйлеу браузерлері де кіреді.
all
HTML5-те медиа сұраулар мәндер ретінде көрсетілуі мүмкін.


rel
Attribute <rel> ағымдағы құжат пен сілтеме жасалған файл арасындағы қатынасты анықтайды. Бұл браузерге қосылатын құжатты қалай пайдалану керектігін білу үшін қажет.

Syntax

<link rel="мәні">

Мәні

Әдетті мәні
alternate
Балама түрі, мысалы, жаңалықтар таспасын, мақалалар хабарландыруларын сипаттау үшін XML форматындағы файлға сілтеме жасау үшін қолданылады.
author
Ағымдағы құжаттың немесе мақаланың авторына сілтемені көрсетеді.
help
Анықтама сілтемесін көрсетеді.
icon
Ағымдағы құжатты немесе сайтты бейнелейтін суреттің мекен-жайы.
license
Ағымдағы құжаттың негізгі мазмұны көрсетілген құжатта сипатталған лицензия бойынша таратылатынын хабарлайды.
next
Ағымдағы құжат өзара байланысты құжаттардың бөлігі болып табылады және сілтеме келесі құжатқа нұсқайды.
prefetch
Ағымдағы беттің немесе бүкіл сайттың алдын ала кэштелген ресурсын көрсетеді.
prev
Ағымдағы құжат өзара байланысты құжаттардың бөлігі болып табылады, ал сілтеме алдыңғы құжатқа нұсқайды.
search
Құжат немесе сайт бойынша іздеу үшін қолданылатын ресурсқа сілтемені көрсетеді.
stylesheet
Қосылатын файл стиль кестесін (CSS) сақтайтынын анықтайды.
Жоқ.


sizes
Көрнекі көрсету үшін белгішелердің өлшемін көрсетеді. Белгішенің өзін браузер мекен-жай жолында көрсету үшін, таңдаулыларға сақталған кезде, сондай-ақ іздеу нәтижелеріне түсінікті болу үшін іздеу жүйелері қолдана алады.

Syntax

<link rel="icon" sizes="еніXбиіктігі">
<link rel="icon" sizes="<ені1>X<биіктігі1> <ені2>X<биіктігі2>* | any">

Мәні

Әдетті мәні
Алдымен белгішенің ені пиксельдермен бірліктерді көрсетпей көрсетіледі (мысалы, 16), содан кейін латын әрпі жазылады х жоғарғы (X) немесе кіші әріппен (x), содан кейін белгішенің биіктігі шығады. Егер файлда бірден бірнеше белгішелер сақталса, олардың өлшемдерін бос орын арқылы орнатуға болады. Кез-келген кілт сөз белгішені кез-келген мөлшерде масштабтауға болатындығын көрсетеді, мысалы, егер ол SVG векторлық форматында сақталса. Жоқ.


type
Сыртқы құжат үшін қандай MIME деректер түрі қолданылатынын браузерге хабарлайды. Әдетте, ол қосылатын файлда CSS бар екенін көрсету үшін қолданылады.

Syntax

<link type="мәні">

Мәні

Әдетті мәні
Кез келген регистрдегі MIME-түрінің атауы. Қосылған стиль кестелері үшін text/css түрі қолданылады. text/css
