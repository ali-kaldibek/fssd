</optgroup> Tag ( option group — "параметрлер тобы" )
бұл бір топқа біріктірілген </option> элементтері орналасқан контейнер.

</optgroup> ерекшелігі-бұл тізімнің қарапайым элементі ретінде ерекшеленбейді, қалың стильмен ерекшеленеді және осы контейнерге кіретін барлық элементтер бастапқы позициясының оң жағына ауысады.

</optgroup> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


disabled
Топ басқа түспен (әдетте сұр) көрсетілетін және таңдау мүмкін емес болған кезде тізім тобына кіруді бұғаттайды.

Syntax

<optgroup disabled>
    ...
</optgroup>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


label
Тізімге топ тақырыбы ретінде кіретін мәтін. Бұл HTML кодтары жоқ қарапайым мәтін жолы.

Syntax

<optgroup label="мәні">
    ...
</optgroup>

Мәні

Әдетті мәні
Кез келген мәтін жолы. Жоқ.
