</td> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.

colspan
Көлденеңінен біріктірілуі тиіс ұяшықтар санын белгілейді. Бұл атрибут бірнеше жолдан тұратын кестелер үшін мағынасы бар.

Syntax

<td colspan="мәні">
    ...
</td>

Мәні

Әдетті мәні
Кез келген бүтін оң Сан. 0 мәні ұяшықты <colgroup> элементімен біріктірілген барлық ата-аналар тобына таратады. 1000-нан жоғары мәндер қате деп саналады және 1-ге орнатылады. 1

headers
Кесте ұяшықтарын тақырыптармен байланыстыруға мүмкіндік береді. Бұл атрибут сөйлеу браузерлерін пайдаланушыларға кестенің қол жетімділігін арттыруға арналған, қарапайым шолғыштарда headers төлсипатын қолдану нәтижесі байқалмайды.
Ұяшықтарды td немесе th элементіндегі бір ұяшыққа байланыстыру үшін id төлсипаты, ал екінші ұяшық — id мәніне сәйкес келетін мәні бар headers төлсипаты.

Syntax

<td id="мәні">
    ...
</td>
<td headers="мәні">
    ...
</td>

Мәні

Әдетті мәні
Бос орынмен бөлінген бір немесе бірнеше идентификаторлар. Жоқ

rowspan
Тігінен біріктірілуі тиіс ұяшықтар санын белгілейді. Бұл атрибут бірнеше жолдан тұратын кестелер үшін мағынасы бар.

Syntax

<td rowspan="мәні">
    ...
</td>

Мәні

Әдетті мәні
Кез келген бүтін оң Сан. Егер мән 0 ретінде орнатылса, онда ұяшықтар кесте бөлімінің соңына (<thead>, <tbody> немесе <tfoot>) немесе кестенің өзіне біріктіріледі. Максималды рұқсат етілген мән-65534. 1
