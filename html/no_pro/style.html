Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


media
Стиль кестесі арналған Шығыс құрылғысын орнатады. Әр құрылғы үшін сіз өзіңіздің жеке стиліңізді анықтай аласыз, ол құрылғының ерекшелігін ескеріп, оған веб-беттің көрінісін реттейді.

Syntax

<style media="мәні">
    ...
</style>

Мәні

Әдетті мәні

Сөйлеу синтезаторлары, сондай-ақ мәтінді дауыстап шығаруға арналған бағдарламалар. Бұған сөйлеу браузерлері де кіреді.

type
Стиль кестесін дұрыс түсіндіру үшін браузерге қандай синтаксисті қолдану керектігін айтады. Әдетте, браузерлер стильдермен дұрыс жұмыс істейді және бұл атрибут болмаған кезде, контейнердің мазмұнын </style> танымайтын кейбір ескі браузерлер үшін қажет.
HTML 4-те бұл атрибут міндетті болып табылады, HTML5-те оны елемеуге болады.

Syntax

<style type="мәні">
    ...
</style>
