</canvas> Tag ( canvas — "кенеп" )
JS көмегімен әртүрлі нысандарды салуға, суреттерді көрсетуге, оларды түрлендіруге және қасиеттерін өзгертуге болатын аймақ жасайды.

</canvas> элементінің көмегімен суреттер, анимациялар, ойындар және т. б. жасауға болады.

Scripts
</canvas> </noscript> </script>





Syntax

<canvas>
    ...
</canvas>

</canvas> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


height
Кенептің биіктігін анықтайды. Әдепкі-150 пиксель.

Syntax

<canvas height="мәні">
    ...
</canvas>

Мәні

Әдетті мәні
Жоқ. 150px


width
Кенептің енін белгілейді. Әдепкі бойынша 300 пиксель.

Syntax

<canvas width="мәні">
    ...
</canvas>

Мәні

Әдетті мәні
