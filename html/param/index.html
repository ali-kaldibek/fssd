<param> Tag ( parameter — "параметр" )
параметр мәндерін <applet> немесе </object> элементтері арқылы жасалған Java апплеттеріне немесе Веб-бет нысандарына беруге арналған.

Бұл тәсіл HTML құжатының кодында объектінің сипаттамаларын қосымша құрастырусыз өзгертуге мүмкіндік береді. Бір уақытта қолданылатын элементтердің саны <param> біреуден көп болуы мүмкін және олардың әрқайсысы үшін name және value атрибуттары арқылы "атау=мән" жұбы берілген.

Syntax

<param>

<param> жұпсыз tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


name
Кәдімгі HTML құжатында браузер терезесіндегі сілтемені нұқу кезінде ағымдағы құжат жаңасына ауыстырылады. Мазмұнды кадрға бағыттау үшін оның name атрибутымен анықталған ерекше атауы болуы керек. Бұл жағдайда </a> сілтемесін құру кезінде кадр атауы мәні ретінде әрекет ететін target атрибутын көрсету қажет.

Syntax

<param name="мәні">

Мәні

Әдетті мәні
Атауды көрсету үшін таңбалар жиынтығы, оның ішінде сандар мен әріптер қолданылады. Аты бойынша кадрға сілтеме жасағанда, name атрибутындағыдай емлені сақтаңыз. Жоқ.


value
Параметр мәнін орнатады. Бұл атрибут міндетті емес, өйткені нысан үшін name атрибутымен берілген бір атау жеткілікті болуы мүмкін.

Syntax

<param name="мәні" value="мәні">

Мәні

Әдетті мәні
Мән ретінде жолды немесе санды пайдалануға болады. Жоқ.
