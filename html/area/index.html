<area> Tag ( area — "аймақ" )
сілтеме болып табылатын белсенді кескін аймақтарын анықтайды.

Оған қосылған белсенді аудандары бар сурет картамен бірге сурет деп аталады. Мұндай карта қарапайым кескіннен еш айырмашылығы жоқ, бірақ оны әр түрлі пішіндегі көрінбейтін аймақтарға бөлуге болады, мұнда әр аймақ сілтеме ретінде қызмет етеді. <area> элементі аймақтың пішінін, оның өлшемдерін анықтайды, сілтеме жасалатын құжаттың мекенжайын белгілейді. <area> әрқашан аудандардың координаттарын кескінмен байланыстыратын </map> контейнерінде орналасқан.

Бірнеше аймақ бір-біріне сәйкес келуі мүмкін, жоғарыда HTML кодында орналасқан аймақ болады.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.
alt
alt атрибуты кескін аймағына балама мәтін орнатады. Мұндай мәтін браузерде көрсетілмейді, бірақ іздеу жүйелері оны оқи алады.

Syntax

<area alt="мәні">

Мәні

Әдетті мәні
Кез келген қолайлы мәтін жолы. Жоқ.
coords
coords атрибуты аймақтың координаттарын белгілейді, оны "ыстық аймақ"деп те атайды. Мұндай аймақ файлға сілтеме болуы мүмкін немесе сценариймен анықталған әрекетке байланысты болуы мүмкін.
Координаталық мәндер үтірмен бөлінген сандар жиынтығы. Егер екі аймақ бір-біріне сәйкес келсе, жоғарыдағы HTML кодында анықталған басымдыққа ие болады.

Syntax

<area coords="мәні">

Мәні

Әдетті мәні
Координаттар жиынтығы shape атрибутымен берілген "ыстық аймақ" пішінімен анықталады. Координаталарды санау әдетте кескіннің сол жақ жоғарғы бұрышынан алынады және пиксельмен көрсетіледі.
Тіктөртбұрыш үшін (shape="rect") 1.суретте көрсетілгендей төрт координат анықталады — X1, Y1, X2, Y2.
Шеңбер үшін (shape="circle") үш координат анықталады — шеңбердің центрінің координаттары (X, Y) және оның радиусы (R) 2.суретте көрсетілгендей.
Полигон (көпбұрыш) үшін (shape="poly") 3.суретте көрсетілгендей әр шыңның координаттары (X1, Y1, X2, Y2,:) рет-ретімен көрсетіледі. 		Жоқ.
href
href атрибуты ( hypertext reference — "гипермәтіндік сілтеме" ) өту керек құжаттың мекенжайын көрсетеді. Кез-келген құжат сілтеме мекенжайы ретінде пайдаланылуы мүмкін болғандықтан, сілтеме бойынша өту нәтижесі соңғы файлға байланысты болады. Сонымен, мұрағаттар (zip немесе rar кеңейтімі бар файлдар) жергілікті дискіге сақталады. Әдепкі бойынша, жаңа құжат ағымдағы шолғыш терезесіне жүктеледі, бірақ бұл сипатты target атрибут арқылы өзгертуге болады.

Syntax

<area href="адрес">

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.

hreflang атрибуты ( hypertext reference language — "гипермәтіндік сілтеме тілі" ) сілтеме жүргізетін құжаттың тілін көрсетеді.

Syntax

<area hreflang="мәні">

Әдепкі бойынша, сілтемені басқан кезде құжат ағымдағы шолғыш қойындысында ашылады. Бұл мінез-құлықты target атрибутының көмегімен өзгертуге болады ( target — "мақсат" ) </a> элементі. Мән ретінде </iframe> элементінің name атрибутымен немесе сақталған кілт сөзбен берілген кадр атауы қолданылады. Кадр атауын көрсеткен кезде сілтеме кадрда ашылады.

type атрибуты ( type — "тип" ) сілтеме көрсететін құжаттың MIME түрін орнатады. Бұл атрибут ұсынымдық сипатқа ие және оны берілген құжат түрімен сілтемелерді сәндеу үшін пайдалануға болады. type төлсипатын тек href төлсипаты болған кезде қосу керек.
