</object> Tag ( object — "объект" )
браузерге бастапқы шолғыш түсінбейтін нысандарды қалай жүктеу және көрсету керектігін хабарлайды.

Әдетте, мұндай нысандар браузерге плагин деп аталатын арнайы модульді қосуды немесе көмекші бағдарламаны іске қосуды қажет етеді.

Сонымен қатар, </object> контейнерінің ішіне нысанды көрсету үшін қосымша параметрлерді беретін <param> элементін орналастыруға болады.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


data
Браузер терезесінде көрсетілетін файлды анықтайды. Танымал деректер форматтары үшін нәтижені жүктеу және көрсету үшін файлдың жолын және оның түрін (type атрибуты) көрсету жеткілікті.
Жол ағымдағы құжатқа қатысты орнатылуы керек.

Syntax

<object data="мәні">
    ...
</object>

Мәні

Әдетті мәні
Файлға толық немесе салыстырмалы жол мән ретінде қабылданады. Жоқ.



height
height Атрибут бейнені ойнату үшін аймақтың биіктігін орнатады. Бейненің өзі берілген кадрға сәйкес келу үшін өлшемдерін үлкен немесе кіші етіп өзгертеді, бірақ оның пропорциялары өзгеріссіз қалады.

Syntax

<embed height="мәні">
    ...
</embed>

Мәні

Әдетті мәні
Пиксельдегі немесе пайыздардағы кез-келген оң Сан. Бастапқы биіктік бейне параметрлерінен алынады. Егер бұл параметрлер қол жетімді болмаса, онда height poster атрибутымен берілген суреттің биіктігіне тең қабылданады. Әйтпесе, бейненің биіктігі 150 пиксельге орнатылады.



type
type атрибуты ( type — "тип" ) сілтеме көрсететін құжаттың MIME түрін орнатады. Бұл атрибут ұсынымдық сипатқа ие және оны берілген құжат түрімен сілтемелерді сәндеу үшін пайдалануға болады. type төлсипатын тек href төлсипаты болған кезде қосу керек.

Syntax

<embed href="мәні">
    ...
</embed>

Мәні

Әдетті мәні
MIME түрі Жоқ.



width
Width төлсипаты бейнені ойнату үшін аймақтың енін белгілейді. Бейненің өзі көрсетілген енге сәйкес келу үшін өлшемдерін үлкен немесе кіші етіп өзгертеді, бірақ оның пропорциялары бұрмаланбайды.

Syntax

<embed width="мәні">
    ...
</embed>

Мәні

Әдетті мәні
Пиксельдегі немесе пайыздардағы кез-келген оң Сан. Бастапқы ені бейне параметрлерінен алынады, егер бұл мән болмаса, онда width poster атрибутымен берілген суреттің еніне тең болады. Егер бұл атрибут орнатылмаса, ені 300 пиксель ретінде орнатылады.
