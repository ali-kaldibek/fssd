</option> Tag ( a — "опция, параметр" )
</select> контейнері арқылы жасалған тізімнің жеке тармақтарын анықтайды.

Тізімнің ені </option> - да көрсетілген ең кең мәтінмен анықталады және стильдер арқылы да өзгеруі мүмкін. Егер сіз тізім деректерін серверге жіберуді жоспарласаңыз, онда форманы </select> ішіне қою керек. Бұл тізім деректеріне сценарийлер арқылы жүгінген кезде де қажет.

</option> жұпты tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


disabled
Таңдау үшін тізім элементін блоктайды.

Syntax

<option disabled>
    ...
</option>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


label
Атрибут </option> ішіндегі мәтінмен салыстырғанда қысқартылған тізім тармағының белгісін көрсетуге арналған. Егер label төлсипаты бар болса, онда </option> ішіндегі мәтін еленбейді және label мәні тізімде көрсетіледі.

Syntax

<option label="мәні">
    ...
</option>

Мәні

Әдетті мәні
Кез келген мәтін жолы. Жоқ.


selected
Тізімнің ағымдағы тармағын ерекшелейді. Егер </select> элементіне multiple төлсипаты қосылса, онда тізімнің бірнеше тармағын таңдауға болады.

Syntax

<option selected>
    ...
</option>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


value
Серверге жіберілетін тізім тармағының мәнін анықтайды. Серверге "атау=мән" жұбы жіберіледі, онда атау элементтің атауы </select> төлсипатымен, ал мән таңдалған тізім тармақтарының value төлсипатымен беріледі. Мән элемент мәтінімен сәйкес келуі мүмкін немесе тәуелсіз болуы мүмкін. Сондай-ақ, value атрибуты сценарийлер арқылы деректер мәндерін алу үшін қолданылады.

Syntax

<option value="мәні">
    ...
</option>
