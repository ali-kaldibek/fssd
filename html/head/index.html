</head> Tag ( head — "бас" )
браузерге деректермен жұмыс істеуге көмектесетін басқа элементтерді сақтауға арналған.

Сондай-ақ </head> контейнерінің ішінде браузерлер мен іздеу жүйелеріне арналған ақпаратты сақтау үшін пайдаланылатын мета тегтер бар. Мысалы, іздеу жүйесінің механизмдері сайт сипаттамасын, кілт сөздерді және басқа деректерді алу үшін мета тегтерге жүгінеді.

Контейнердің ішіне </head> келесі элементтерді орналастыруға рұқсат етіледі: <base>, <link>, <meta>, </script>, </style>, </title>.

</head> мазмұны тікелей веб-бетте көрсетілмейді, </title> элементінен басқа, ол веб-бет терезесінің тақырыбын орнатады.
