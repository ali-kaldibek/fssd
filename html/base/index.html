<base> жұпсыз tag

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.
href
Файлдардың толық жолын көрсету үшін пайдаланылатын мекен-жай. Бұл әдетте ағымдағы құжаттың әдеттегі жолы, бірақ сайтта файлдарды ұйымдастыру қажет болса, оны басқаларға да орнатуға болады.

Syntax

<base href="мәні">;

Мәні

Әдетті мәні
Құжатқа жол. Ағымдағы файлға жол.
target
Барлық сілтемелер ашылатын жақтаудың атын анықтайды. Атау <iframe> элементінің name атрибутымен анықталады. Сондай-ақ, жаңа қойындыда сілтемелерді ашатын кілт сөзді мағынасы ретінде пайдалануға болады.

Syntax

<base target="мәні">

Мәні

Әдетті мәні
_blank
Бетті жаңа шолғыш қойындысына жүктейді.
_self
Бетті ағымдағы қойындыға жүктейді.
_parent
Бетті ата-ана жақтауына жүктейді; егер жақтаулар болмаса, онда бұл мән <_self> ретінде жұмыс істейді.
_top
Барлық Жақтауларды болдырмайды және бетті толық шолғыш терезесінде жүктейді; егер жақтаулар болмаса, онда бұл мән <_self> ретінде жұмыс істейді.
