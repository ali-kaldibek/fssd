<input> Tag ( input — "енгізу" )
пішіннің жан-жақты элементтерінің бірі болып табылады және интерфейстің әртүрлі бөліктерін құруға және пайдаланушы тәжірибесін қамтамасыз етуге мүмкіндік береді.

Негізінен <input> мәтін өрістерін, әртүрлі түймелерді, қосқыштарды және құсбелгілерді жасауға арналған.

Элемент түрін анықтайтын <input> негізгі төлсипаты-type. Ол Сізге келесі пішін элементтерін орнатуға мүмкіндік береді: мәтін өрісі (text), пароль өрісі (password), қосқыш (radio), құсбелгі, тексеру жәшігі (checkbox), жасырын өріс (hidden), батырма (button), пішінді жіберу түймесі (submit), пішінді тазарту түймесі (reset), файлды жіберу өрісі (file), кескін түймесі (image) және т.б. әр элемент үшін оның түрі мен сипаттамаларын анықтайтын атрибуттардың жеке тізімі бар. Сонымен қатар, HTML5-те оннан астам жаңа атрибуттар қосылды.

Syntax

<input> жұпсыз tag



Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


accept
Файлды жүктеу өрісі арқылы жіберуге болатын файл түрлеріне сүзгіні орнатады. Файл түрі MIME түрі ретінде көрсетіледі, бірнеше мәндерде олар үтірмен тізімделеді. Егер файл орнатылған сүзгіге сәйкес келмесе, ол файл таңдау терезесінде көрсетілмейді.
Файлды жіберу үшін өріске қолданылады (<input type="file">).

Syntax

<input type="file" accept="MIME-түріні">

Мәні

Әдетті мәні
Кез келген регистрдегі MIME түрінің атауы, бірнеше мәндер үтір арқылы тізімделеді.
HTML5-те барлық дыбыстық файлдарды таңдау үшін audio/*, бейне файлдар үшін video/*> және барлық графикалық файлдар үшін image/* мәні ретінде көрсетуге болады.
MIME файл түрлерінің тізімі. 		Жоқ.


alt
alt атрибуты кескін өрісі үшін балама мәтінді орнатады. Мұндай мәтін браузерде суреттерді жүктеу кезінде сурет туралы мәтіндік ақпаратты алуға мүмкіндік береді, сонымен қатар іздеу жүйелеріне арналған.

Syntax

<input type="image" alt="мәні">

Мәні

Әдетті мәні
Кез келген қолайлы мәтін жолы. Жоқ.


autocomplete
Бұл атрибут пішін өрістерін бұрын енгізілген мәтінмен толтыруға көмектеседі. Мәндер браузерді сақтайды және ауыстырады, ал қауіпсіздік мақсатында автотолтыру браузер параметрлерінде өшірілуі мүмкін және бұл жағдайда autocomplete атрибутымен басқарыла алмайды.
Мәтіннің бірінші әріптерін енгізген кезде, сіз қажетті мәндерді таңдауға болатын бұрын сақталған мәндердің тізімі көрсетіледі.
Пішіннің нақты өрісін автотолтыру оның атрибутына қосылады және мән өзгерген кезде жоғалады.

Syntax

<input type="text" autocomplete="мәні">
<input type="password" autocomplete="мәні">
<input type="email" autocomplete="мәні">
<input type="search" autocomplete="мәні">
<input type="url" autocomplete="мәні">
<input type="tel" autocomplete="мәні">

Мәні

Әдетті мәні
on
Мәтінді автоматты түрде толтыруды қамтиды.
off
Автотолтыруды өшіреді. Бұл мән әдетте браузерде маңызды деректерді (парольдер, банк карталарының нөмірлері), сондай-ақ сирек енгізілетін немесе бірегей деректерді (captcha) сақтауды болдырмау үшін қолданылады.
Браузер параметрлеріне байланысты.


autofocus
Пішін өрісіне фокусты автоматты түрде орнатады. Бұл өрісте сіз оны тінтуірдің курсорымен анық баспай-ақ бірден тере аласыз.

Syntax

<input autofocus>

Әдетті мәні
Жоқ. Әдепкі бойынша, autofocus төлсипаты орнатылмаған.


checked
Бұл атрибут құсбелгі немесе қосқыш сияқты пішін элементінің алдын-ала белгіленгенін анықтайды. Қосқыштарды қолданған жағдайда (radiobutton) топтың тек бір элементін белгілеуге болады, құсбелгілер үшін (checkbox) кем дегенде барлық элементтерді белгілеуге рұқсат етіледі.

Syntax

<input type="radio" checked>
<input type="checkbox" checked>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


dirname
Егер dirname атрибуты болса, серверге атау/мән жұбы жіберіледі, онда атау dirname атрибутымен көрсетіледі, ал мән браузермен ауыстырылады. Солдан оңға қарай мәтін үшін мән ltr, ал оңнан солға қарай мәтін үшін (мысалы, Иврит) мән rtl болады.

Syntax

<input type="search" dirname="строка">

Мәні

Әдетті мәні
Произвольная текстовая строка, которая служит именем параметра. Чтобы показать, что этот параметр указывает направление текста, обычно в конце строки добавляют .dir. Жоқ.


disabled
Пішін өрісіне кіруді және өзгертуді блоктайды. Бұл жағдайда ол сұр болып көрінеді және оны іске қосу үшін пайдаланушы қол жетімді емес. Сонымен қатар, мұндай өріс Tab пернесін, тінтуірді немесе басқа әдісті басу арқылы фокусты ала алмайды. Дегенмен, өрістің бұл күйін сценарийлердің көмегімен өзгертуге болады. Өрістегі бұғатталған мән серверге берілмейді.

Syntax

<input disabled>

form
Өрісті оның идентификаторы бойынша формамен байланыстырады. Мұндай байланыс өріс form шегінен тыс орналасқан жағдайда, мысалы, оны бағдарламалық түрде немесе дизайн себептері бойынша жасаған кезде қажет.

Syntax

<input type="тип" form="идентификатор">
<form id="идентификатор">...</form>

Әдетті мәні
Пішін идентификаторы (<form> элементінің id төлсипатының мәні). Жоқ.


formaction
Пішін өңдегішінің мекен — жайын анықтайды-бұл пішін деректерін алатын және олармен қажетті әрекеттерді жасайтын бағдарлама. formaction атрибутының әрекеті </form> элементінің action атрибутына ұқсас.

Syntax

<input type="submit" formaction="адрес">

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


formenctype
Серверге жіберілген кезде форма деректерін кодтау әдісін орнатады. Әдетте файлды жіберу үшін өріс қолданылған жағдайда нақты көрсетіледі (input type="file"). Бұл атрибут </form> элементінің enctype атрибутына ұқсас, formenctype және enctype ортақ пайдалану кезінде соңғысы еленбейді.

Syntax

<input type="submit" formenctype="мәні">

Мысалы, толық мекен-жайы болады http://site.ru/doc/?name=Vasya&password=pup. әдістегі деректер көлемі 4 Кб-мен шектелген.

Серверге браузер сұрауында деректерді жібереді, жіберілетін деректердің көлемі тек сервер баптауларымен шектеледі.
get

Syntax

Фокусты алған кезде немесе теру кезінде жоғалып кететін пішін өрісіндегі мәтінді көрсетеді. Әдетте сұр түспен көрсетіледі.

Syntax

<input placeholder="мәні">

Мәні

Әдетті мәні
Мәтін жолы. Егер жол ішінде бос орын болса, оны екі немесе бір тырнақшаға алу керек. Жоқ.


readonly
<input> элементіне readonly төлсипаты қосылған кезде, пайдаланушы мәтін өрісін өзгерте алмайды, оның ішінде жаңа мәтін енгізілмейді немесе бар мәтін өзгертілмейді. Алайда, өрістің күйі мен мазмұнын сценарийлер арқылы өзгертуге болады және деректер серверге жіберіледі.

Syntax

<input type="text" readonly>
<input type="password" readonly>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


required
Нысанды серверге жібермес бұрын толтыру үшін міндетті форма өрісін орнатады. Егер міндетті өріс бос болса, шолғыш қате туралы хабарлама шығарады және форма жіберілмейді. Хабарламаның түрі мен мазмұны браузерге байланысты және пайдаланушы өзгерте алмайды.

Syntax

<input required>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша, required атрибут өшірулі.


size
Моно-шрифт таңбаларының санымен анықталатын мәтін өрісінің ені. Басқаша айтқанда, ені көлденең ені бірдей жақын әріптердің санымен анықталады. Егер қаріп өлшемі стильдермен өзгертілсе, ені де сәйкесінше өзгереді.

Syntax

<input type="text" size="мәні">
<input type="password" size="мәні">

Мәні

Әдетті мәні
Кез келген бүтін оң Сан. 20


src
Кескін өрісіндегі веб-бетте көрсетілетін графикалық файлдың мекен-жайы. PNG және JPEG форматындағы ең танымал файлдар.

Syntax

<input type="image" src="адрес">

Мәні

Әдетті мәні
Файлға толық немесе салыстырмалы жол мән ретінде қабылданады. Жоқ.


step
Жүгірткілер мен сандарды енгізу өрістері үшін санды өзгерту қадамын орнатады.

Syntax

<input type="number" step="мәні">
<input type="range" step="мәні">

Мәні

Әдетті мәні
Кез келген бүтін немесе бөлшек сан. 1


type
Пішін элементі қай түрге жататынын браузерге хабарлайды.

Syntax

<input type="мәні">

Мәні

Әдетті мәні

type мәндері:

button
Түйме.
checkbox
Жалаушалар. Ұсынылған нұсқалардан бірнеше нұсқаны таңдауға мүмкіндік береді.
file
Серверге жіберілетін файл атауын енгізу өрісі.
hidden
Жасырын өріс. Ол веб-бетте көрсетілмейді.
image
Суреті бар өріс. Суретті басқан кезде бұл пішіндер серверге жіберіледі.
password
Кәдімгі мәтін өрісі, бірақ одан барлық таңбалар жұлдызшалармен көрсетілетіндігімен ерекшеленеді. Енгізілген парольді ешкім көрмейтіндей етіп жасалған.
radio
Қосқыштар. Ұсынылған бірнеше нұсқаның біреуін таңдаған кезде қолданылады.
reset
Пішін деректерін бастапқы мәнге қайтаруға арналған батырма.
submit
Пішін деректерін серверге жіберуге арналған батырма.
text
Мәтін өрісі. Пернетақтаны пайдаланып таңбаларды енгізуге арналған.
HTML5 Жаңа мәндерді қосты:
color
Түсті таңдауға арналған Виджет.
date
Күнтізбелік күнді таңдау өрісі.
datetime
Күні мен уақытын көрсету.
datetime-local
Жергілікті күн мен уақытты көрсету.
email
Электрондық пошта мекенжайлары үшін.
number
Іске қосу сандар.
range
Көрсетілген диапазондағы сандарды таңдауға арналған жүгірткі.
search
Іздеу өрісі.
tel
Телефон нөмірлері үшін.
time
Уақыт үшін.
url
Веб-мекенжайлар үшін.
month
Айды таңдау.
week
Апта таңдау.
text


value
Серверге жіберілетін немесе клиенттік сценарийлер арқылы алынатын пішін элементінің мәнін анықтайды. Серверге "атау=мән" жұбы жіберіледі, онда атау <input> элементінің name төлсипатымен, ал value төлсипатымен беріледі.
Элемент түріне байланысты value атрибуты келесі рөл атқарады:
батырмалар үшін (input type="button | reset | submit") оларға мәтіндік жазба орнатады;
Мәтін өрістері үшін (input type="password | text") алдын-ала енгізілген жолды көрсетеді. Пайдаланушы мәтінді өшіре алады және оның таңбаларын енгізе алады, бірақ Reset батырмасы түрінде қолданылған кезде, value төлсипатына енгізілген жеке мәтін тазаланады және қалпына келтіріледі;
ұяшықтар мен қосқыштар үшін (input type="checkbox | radio") әр элементті бірегей түрде анықтайды, осылайша клиент немесе сервер бағдарламасы пайдаланушының қай элементті таңдағанын анықтай алады.
файл өрісі үшін (input type="file") әсер етпейді.

Syntax

<input value="мәні">
