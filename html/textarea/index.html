</textarea> Tag ( text area — "мәтін аймағы" )
бұл мәтіннің бірнеше жолын енгізуге болатын аймақты құру үшін пішін элементі. Өріске енгізуге болатын таңбалар саны шектелмейді, ал енгізілген мәтін моно-шрифтпен көрсетіледі.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.



autocomplete
Бұл атрибут мәтін өрісін бұрын енгізілген мәтінмен толтыруға көмектеседі. Мәндер браузерді сақтайды және ауыстырады, ал қауіпсіздік мақсатында автотолтыру браузер параметрлерінде өшірілуі мүмкін және бұл жағдайда autocomplete атрибутымен басқарыла алмайды.
Пішіннің нақты өрісін автотолтыру оның атрибутына қосылады және мән өзгерген кезде жоғалады.

Syntax

<textarea autocomplete="мәні">
    ...
</textarea>

Мәні

Әдетті мәні
on
Мәтінді автоматты түрде толтыруды қамтиды.
off
Автотолтыруды өшіреді.
Браузер параметрлеріне байланысты.


autofocus
Фокусты мәтін өрісіне автоматты түрде орнатады. Бұл өрісте жүгіргіні мәтін өрісіне ауыстырмай бірден теруге болады.

Syntax

<textarea autofocus>
    ...
</textarea>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша, autofocus төлсипаты орнатылмаған.


cols
Моно-шрифт таңбаларының санымен анықталатын мәтін өрісінің ені. Басқаша айтқанда, ені көлденең ені бірдей жақын әріптердің санымен анықталады. Егер қаріп өлшемі стильдермен өзгертілсе, ені де сәйкесінше өзгереді.

Syntax

<textarea cols="мәні">
    ...
</textarea>

Әдетті мәні

dirname
Егер dirname атрибуты болса, серверге атау/мән жұбы жіберіледі, онда атау dirname атрибутымен көрсетіледі, ал мән браузермен ауыстырылады. Солдан оңға қарай мәтін үшін мән ltr> болады, ал оңнан солға мәтін үшін (мысалы, арабша) мән rtl болады.

Syntax

<textarea dirname="строка">
    ...
</textarea>

Мәні

Әдетті мәні
Параметр атауы ретінде қызмет ететін еркін мәтін жолы. Бұл параметр мәтіннің бағытын көрсететінін көрсету үшін, әдетте, жолдың соңында қосылады .dir. Жоқ.


disabled
Мәтін өрісіне кіруді және өзгертуді блоктайды. Бұл жағдайда ол сұр болып көрінеді және оны іске қосу үшін пайдаланушы қол жетімді емес. Сонымен қатар, мұндай өріс Tab пернесін, тінтуірді немесе басқа әдісті басу арқылы фокусты ала алмайды. Дегенмен, өрістің бұл күйін сценарийлердің көмегімен өзгертуге болады.
Өрістегі бұғатталған мәтін серверге берілмейді.

Syntax

<textarea disabled>
    ...
</textarea>

Syntax

Мәні

Әдетті мәні
Атау ретінде таңбалар жиынтығы, оның ішінде сандар мен әріптер қолданылады. JS регистрге сезімтал, сондықтан элементке сілтеме жасаған кезде </textarea> аты бойынша name атрибутымен бірдей жазу формасын сақтаңыз. Жоқ.

Фокусты алу немесе теру кезінде жоғалып кететін мәтін ұясының ішіндегі мәтінді көрсетеді. Кеңес ретінде әдетте бір сөзден немесе сөйлемнен тұратын қысқа мәтін қолданылады.

    ...

Әдетті мәні

</textarea> элементіне readonly атрибуты қосылған кезде, пайдаланушы мәтін өрісін өзгерте алмайды, оның ішінде жаңа мәтін енгізілмейді немесе бар мәтін өзгертілмейді. Дегенмен, өрістің күйі мен мазмұнын сценарийлер арқылы өзгертуге болады. Сондай-ақ, өрістің мазмұнын таңдап, алмасу буферіне көшіруге болады.

Syntax

<textarea readonly>
    ...
</textarea>

Әдетті мәні

wrap
wrap төлсипаты шолғышқа </textarea> өрісіне мәтінді қалай беру керектігін және серверге деректерді қалай жіберу керектігін айтады. Егер бұл атрибут болмаса, өрістегі мәтін бір жолмен теріледі, енгізілген таңбалар саны аймақтың енінен асып кетсе, көлденең айналдыру жолағы пайда болады. Enter батырмасын басу мәтінді жаңа жолға апарады және курсор өрістің сол жақ шетіне орнатылады.

Syntax

<textarea wrap="мәні">
    ...
</textarea>
