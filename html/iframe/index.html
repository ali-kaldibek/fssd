</iframe> Tag ( inline frame — "енгізілген жақтау" )
кәдімгі құжаттың ішінде орналасқан кірістірілген жақтауды жасайды, ол кез-келген басқа тәуелсіз құжаттарды берілген өлшем аймағына жүктеуге мүмкіндік береді.

<iframe> мазмұны осы элементті қолдамайтын браузерлер елемейтін контейнер болып табылады. Мұндай браузерлер үшін пайдаланушылар көретін балама мәтінді көрсетуге болады. Ол <iframe> және </iframe> тегтерінің арасында орналасуы керек.

Attributes

Бұл элемент үшін әмбебап атрибуттар қол жетімді.


allowfullscreen
allowfullscreen атрибуты кадрға толық экран режиміне мүмкіндік береді.

Syntax

<iframe allowfullscreen>
    ...
</iframe>

Мәні

Әдетті мәні
Жоқ. Әдепкі бойынша бұл мән өшірулі.


height
height Атрибут бейнені ойнату үшін аймақтың биіктігін орнатады. Бейненің өзі берілген кадрға сәйкес келу үшін өлшемдерін үлкен немесе кіші етіп өзгертеді, бірақ оның пропорциялары өзгеріссіз қалады.

Syntax

<embed height="мәні">
    ...
</embed>

Мәні

Әдетті мәні
Пиксельдегі немесе пайыздардағы кез-келген оң Сан. Бастапқы биіктік бейне параметрлерінен алынады. Егер бұл параметрлер қол жетімді болмаса, онда height poster атрибутымен берілген суреттің биіктігіне тең қабылданады. Әйтпесе, бейненің биіктігі 150 пиксельге орнатылады.


name
Кәдімгі HTML құжатында браузер терезесіндегі сілтемені нұқу кезінде ағымдағы құжат жаңасына ауыстырылады. Мазмұнды кадрға бағыттау үшін оның name атрибутымен анықталған ерекше атауы болуы керек. Бұл жағдайда </a> сілтемесін құру кезінде кадр атауы мәні ретінде әрекет ететін target атрибутын көрсету қажет.

Syntax

<iframe name="мәні">
    ...
</iframe>

Мәні

Әдетті мәні
Атауды көрсету үшін таңбалар жиынтығы, оның ішінде сандар мен әріптер қолданылады. Аты бойынша кадрға сілтеме жасағанда, name атрибутындағыдай емлені сақтаңыз. Жоқ.


sandbox
sandbox атрибуты кадрға жүктелетін мазмұнға бірқатар шектеулер қоюға мүмкіндік береді, мысалы, пішіндер мен сценарийлерді бұғаттайды. Бұл ағымдағы құжаттың қауіпсіздігін арттыруға мүмкіндік береді, әсіресе құжат жақтауға тексерілмеген көзден жүктелген жағдайда.

Syntax

<iframe src="URL" sandbox="мәні">
    ...
</iframe>

Әдетті мәні
allow-forms
Кадр мазмұны пішіндерді жіберуге мүмкіндік береді.
allow-modals
JS-тегі alert функциясы арқылы жасалатын модальды терезелерді ашуға мүмкіндік береді.
allow-orientation-lock
Экранды құлыптау мүмкіндігін өшіруге мүмкіндік береді.
allow-pointer-lock
Жақтауға Pointer lock API қолдануға мүмкіндік береді.
allow-popups
Қалқымалы терезелерге рұқсат береді (мысалы, window.open, target="_blank", showModalDialog).
allow-presentation
Фреймге Presentation API қолдануға рұқсат береді.
allow-same-origin
Шешеді жүктеу мазмұны жақтаудың, воспринимая оның сол көзінің және ата-аналар құжат. Қалқымалы терезелерді бұғаттау кезінде мазмұнды қауіпсіз ашу үшін пайдалануға болады.
allow-scripts
Сценарийлерді іске қосуға және орындауға мүмкіндік береді. Құру қалқып шығатын терезелер бұл ретте тыйым салынады.
allow-top-navigation
Ата-ана құжатындағы кадр сілтемелерін ашуға мүмкіндік береді.
Жоқ.


scrolling
Егер кадрдың мазмұны белгіленген өлшемдерге сәйкес келмесе, ақпаратты қарау үшін айналдыру жолақтары автоматты түрде пайда болады. Кейбір жағдайларда айналдыру жолақтары веб-беттің дизайнын бұзады, сондықтан оларды тастауға болады. Айналдыру жолақтарының дисплейін басқару үшін scrolling төлсипаты қолданылады.

Syntax

<iframe scrolling="мәні">
    ...
</iframe>

Әдетті мәні

no
Айналдыру жолақтарын көрсетуге тыйым салады.
yes
Әрқашан ақпарат көлеміне қарамастан айналдыру жолақтарының пайда болуына себеп болады.

Syntax

<iframe seamless>

</iframe>

Ойнатылатын аудио файлдың жолын көрсетеді. Осы мақсатта <source> элементін де пайдалануға болады.

</embed>

Мәні

Әдетті мәні
Файлға толық немесе салыстырмалы жол мән ретінде қабылданады. Жоқ.


srcdoc
Кадрдың мазмұнын тікелей атрибутқа орнатуға мүмкіндік береді. Мән дұрыс HTML синтаксисіне ие болуы керек, қажет болса <!DOCTYPE> және </html>, сонымен қатар кез-келген бос орын, жол тасымалы, түсініктемелер және басқа элементтер. src және srcdoc атрибуттарын бір уақытта қолданған кезде src атрибуты еленбейді.

Syntax

<iframe srcdoc="мәні">
    ...
</iframe>

Syntax

    ...
